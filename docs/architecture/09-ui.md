> Web 管理面板，提供 Gateway 的可视化管理界面。

## 功能模块

| 模块 | 视图 | 说明 |
|------|------|------|
| **总览** | `overview*.ts` | Gateway 状态仪表盘，显示运行状态、事件日志 |
| **会话** | `sessions.ts` | 会话列表与管理，支持查看历史、发送消息 |
| **技能** | `skills*.ts` | 技能浏览、安装、配置 |
| **用量** | `usage*.ts` | Token 使用量、成本统计、趋势图表 |
| **节点** | `nodes*.ts` | 移动端/桌面端节点管理 |
| **日志** | `logs.ts` | 实时日志查看 |
| **登录** | `login-gate.ts` | 认证入口 |

## 技术栈

- **构建**：Vite
- **测试**：Vitest（含浏览器测试）
- **截图测试**：`__screenshots__/` 目录下的视觉回归测试
