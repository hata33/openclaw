# 34 — 国际化（i18n）

> OpenClaw 的国际化模块管理多语言翻译，使系统提示、错误消息、
> CLI 输出等可以按用户语言显示。

## 设计

```
i18n 系统：
  → 注册翻译字符串
  → 根据用户语言偏好选择
  → 支持插值变量
  → 回退到英文（默认）
```

## 语言检测

```
检测优先级：
1. 配置显式指定 (config.language)
2. 系统环境变量 (LANG, LC_ALL)
3. 渠道用户设置
4. 默认英文
```

## 使用场景

```
CLI 输出 → 用户语言
  → 英文: "Gateway started successfully"
  → 中文: "网关启动成功"

系统提示 → Agent 语言适配
  → 根据用户语言调整 system prompt

错误消息 → 多语言错误描述
  → 帮助用户理解错误
```

## 关键代码入口

| 文件 | 职责 |
|------|------|
| `src/i18n/registry.ts` | 翻译注册表 |

## 总结

1. **多语言支持** — CLI、系统提示、错误消息均可翻译
2. **自动检测** — 根据环境和配置自动选择语言
3. **回退机制** — 未翻译的内容回退到英文
