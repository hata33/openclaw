# 36 — Crestodian 审计助手

> OpenClaw 的 Crestodian 模块是一个内置的审计和合规助手，
> 负责监控 Agent 行为、执行安全审计和生成合规报告。

## 设计思想

```
Agent 拥有强大的能力（执行命令、访问文件、发送消息...）
  → 需要监控和审计其行为
  → 确保不违反安全策略
  → 生成可追溯的操作日志

Crestodian 作为"审计员"角色：
  → 独立于主 Agent 运行
  → 审查敏感操作
  → 记录决策依据
  → 生成审计报告
```

## 核心功能

### 审计引擎

`src/crestodian/audit.ts` — 执行安全审计：

```
审计触发
  → 检查 Agent 配置安全性
  → 检查工具权限是否合理
  → 检查渠道配置安全性
  → 生成审计报告
  → 发现风险 → 提示修复建议
```

### 助手系统

`src/crestodian/assistant.ts` — Crestodian 的助手角色：

```
Crestodian 本身也是一个 Agent
  → 使用专门的提示词 (assistant-prompts.ts)
  → 可配置不同的后端模型 (assistant-backends.ts)
  → 独立运行，不干扰主 Agent
```

### 配置审计

```
检查项：
  → 密码是否设置
  → DM 访问控制是否合理
  → 工具权限是否过于宽松
  → 模型配置是否安全
  → 渠道凭证是否安全存储
```

## 总结

1. **内置审计** — 自动检查配置和操作安全性
2. **独立运行** — 不干扰主 Agent，作为审计员角色
3. **配置检查** — 发现安全风险并提供修复建议
4. **审计报告** — 生成可追溯的操作日志
