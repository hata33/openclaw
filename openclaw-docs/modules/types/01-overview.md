# types — 功能定义

## 解决什么问题？

TypeScript 项目需要类型声明。某些第三方库没有自带类型声明，也没有社区维护的 @types 包。types 模块为这些库提供本地类型声明。

## 覆盖的库

| 库 | 用途 |
|------|------|
| `create-markdown-preview` | Markdown 预览 |
| `node-pty` | 伪终端（PTY） |
| `@microsoft/teams-sdk` | Teams 集成 |
| `@modelcontextprotocol/sdk` | MCP 协议 |
| `node-edge-tts` | Edge TTS 语音 |
| `node-llama-cpp` | 本地 LLM 推理 |
| `pdfjs-dist` | PDF 解析 |
| `pi-agent-core` | Agent 核心库 |
| `pi-coding-agent` | 编码 Agent |
| `qrcode` | QR 码生成 |
| `web-push` | Web Push 通知 |
