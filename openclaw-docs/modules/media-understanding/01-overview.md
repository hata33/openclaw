# media-understanding — 功能定义

## 解决什么问题？

用户发送多媒体内容时，Agent 需要理解这些内容才能有效回复。

## 图片理解

- 图片描述（场景、物体、文字）
- OCR（图片中的文字识别）
- 图片分析（表情、图表）

## 音频理解

- 语音转文字（Whisper）
- 音频描述

## 视频理解

- 视频帧提取 + 图片理解
- 视频转录

## 文档理解

- PDF 内容提取
- 文档摘要

## Provider 选择

```
1. 配置的 Provider
2. 多模态模型（支持视觉的 LLM）
3. CLI 工具（本地安装的）
4. 回退：提示无法处理
```
