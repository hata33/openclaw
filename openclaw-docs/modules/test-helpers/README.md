# test-helpers — 测试辅助

> 测试环境的辅助工具：HTTP 模拟、临时目录、SSRF 测试等。

## 文件结构

| 文件 | 职责 |
|------|------|
| `http.ts` | HTTP 测试辅助 |
| `network-interfaces.ts` | 网络接口模拟 |
| `resolve-target-error-cases.ts` | 错误用例 |
| `ssrf.ts` | SSRF 测试 |
| `state-dir-env.ts` | 状态目录环境 |
| `temp-dir.ts` | 临时目录 |
| `windows-cmd-shim.ts` | Windows 命令 shim |
| `workspace.ts` | 工作空间辅助 |
