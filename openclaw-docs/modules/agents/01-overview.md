# agents — 架构概览

## 设计思想

### 1. 模型无关

Agent 不绑定特定 AI 模型。通过 Provider Runtime 适配不同模型。

### 2. 工具可扩展

工具通过插件系统注册，支持动态添加。

### 3. 沙箱安全

工具执行受沙箱策略控制，防止危险操作。

### 4. 流式处理

Agent 回复通过流式传输实时显示。

### 5. 上下文压缩

对话历史过长时自动压缩，保留关键信息。

## Agent 类型

| 类型 | 说明 |
|------|------|
| 主 Agent | 处理用户直接消息 |
| 子 Agent | 由主 Agent 生成，执行子任务 |
| CLI Agent | 在终端中运行 |
| Cron Agent | 由定时任务触发 |
