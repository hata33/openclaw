# agents — 核心能力

## 工具类型

| 工具 | 说明 |
|------|------|
| `exec` | 执行 Shell 命令 |
| `fs_read` | 读取文件 |
| `fs_write` | 写入文件 |
| `browser` | 浏览器控制 |
| `memory` | 记忆读写 |
| `cron` | 定时任务 |
| `canvas` | Canvas 展示 |
| `web_search` | Web 搜索 |
| `web_fetch` | 网页抓取 |
| `tts` | 语音合成 |

## 子模块能力

- **auth-profiles**: 多认证配置管理
- **pi-embedded-runner**: 核心模型调用循环
- **sandbox**: 工具执行沙箱
- **skills**: 技能发现和执行
- **tools**: 50+ 内置工具
