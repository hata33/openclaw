# infra — 基础设施

> 373 文件，78948 行。OpenClaw 的基础设施层。
> 提供文件系统、网络、安全、命令分析等底层能力。

## 子模块

| 子目录 | 说明 |
|--------|------|
| `command-analysis/` | 命令安全分析（白名单、内联 eval 检测） |
| `command-explainer/` | 命令解释（人类可读描述） |
| `format-time/` | 时间格式化 |
| `net/` | 网络工具（SSRF 防护、Fetch 包装） |
| `outbound/` | 出站连接管理 |
| `tls/` | TLS/SSL 工具 |

## 核心能力

### 命令安全

分析 Shell 命令的安全性，检测危险操作。

### 网络安全

SSRF 防护、URL 验证、安全 Fetch。

### 设备管理

设备身份、配对认证。

### 进程管理

PID 追踪、进程锁。

### 文件系统

安全文件操作、JSON 文件管理。
